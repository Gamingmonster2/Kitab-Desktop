# Kitab Desktop - المصحف الرقمي

A high-quality Quran reader desktop application built with Python and PySide6.

## Features
- **High Resolution**: Uses high-quality images from the King Fahd Complex (KFGQPC).
- **Multiple Editions**: Support for Hafs, Warsh, and Tajweed.
- **Modern UI**: Dark mode with smooth navigation.
- **Free & Open Source**: Designed for the community.

## Installation
Currently available as a Python script. Standalone `.exe` coming soon in Releases.

### Prerequisites
- Python 3.10+
- PySide6

### Running the App
1. Clone the repository.
2. Run `python assets_downloader.py` to download the images.
3. Run `python reader_ui.py` to start the reader.
